Automatically generates AI-powered summaries for WordPress posts with universal editor support and front-end display.

== Description ==

AI Blog Summary Generator is a comprehensive WordPress plugin that automatically generates AI-powered summaries for your blog posts. It works seamlessly across all WordPress editing environments including Gutenberg Block Editor, Classic Editor, Page Builders, and Custom Post Types.

== Features ==

* Universal Editor Integration
  * Gutenberg/Block Editor sidebar panel
  * Classic Editor meta box
  * Page Builder compatibility (Elementor, Divi, WPBakery, etc.)
  * Custom Post Type support

* AI Summary Generation
  * Multiple AI provider support
  * Configurable summary length
  * Language selection
  * Multiple summary variants

* Front-End Display
  * Lightning-bolt icon beside post titles
  * Beautiful popup modal
  * Smooth animations
  * Light/dark mode support
  * Fully responsive and accessible

* Interaction Tracking
  * Post-based click tracking
  * Global statistics
  * REST API endpoints

* Admin Settings
  * API configuration
  * Model selection
  * Editor preferences
  * Icon customization
  * Popup styling options

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-blog-summary` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Settings > AI Blog Summary to configure your API keys and preferences.
4. Start generating summaries for your posts!

== Frequently Asked Questions ==

= Does this work with all editors? =

Yes! The plugin supports Gutenberg Block Editor, Classic Editor, Page Builders, and Custom Post Types.

= Which AI providers are supported? =

The plugin supports multiple AI providers. You can switch between providers in the settings page.

= Do I need coding knowledge? =

No, the plugin is designed to work out of the box with minimal configuration.
